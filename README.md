Heart Disease Prediction using Machine Learning

This project predicts the likelihood of heart disease using machine learning algorithms.
The complete pipeline includes data cleaning, preprocessing, model training, evaluation, and model saving.

 Features
- Data cleaning & preprocessing
- Handling missing values
- Encoding categorical variables
- Feature scaling
- Multiple ML models comparison
- Model evaluation using accuracy, confusion matrix & classification report
- Saved trained model for deployment

Machine Learning Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Random Forest Classifier
  
Dataset
- HeartDiseaseTrain-Test.csv (raw dataset)
- clean_heart_dataset.csv (cleaned dataset)

Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib
